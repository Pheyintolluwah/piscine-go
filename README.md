# Piscine Go

A collection of Go programming exercises completed during the **01 Edu Piscine Go**.

This repository contains solutions to the Piscine Go challenges, covering the fundamentals of the Go programming language, problem-solving, and algorithmic thinking.

---

## Objectives

- Learn Go syntax and programming fundamentals
- Develop logical thinking and problem-solving skills
- Practice writing clean and efficient code
- Understand algorithms and data structures
- Gain experience with the Go standard library

## Requirements

- Go 1.20 or newer

Check your version:

```bash
go version
```

---

## Clone the Repository

```bash
git clone https://github.com/YOUR_USERNAME/piscine-go.git
```

Move into the repository:

```bash
cd piscine-go
```

---

## Running a Project

Navigate to the exercise folder.

Example:

```bash
cd firstword
```

Run the program:

```bash
go run .
```

or

```bash
go run main.go
```
